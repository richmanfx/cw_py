Запуск программы cw_py.py на питоне в Windows-7
-----------------------------------------------

1.Скачать и установить Python 2.7 с сайта www.python.org/downloads/,
  на момент написания руководства доступна версия 2.7.12 .

2.Добавить пути к питону в переменную среды PATH:
  Пуск -> Компьютер -> правой кнопкой Свойства -> Дополнительные параметры ->
  Переменные среды -> Системные переменные -> Path -> Изменить: в конец строки
  добавить после точки с запятой "C:\Python27;C:\Python27\Scripts;" -> OK ->
  OK -> OK.

3.Открыть окно командной строки: Win+R -> Открыть: -> набрать "CMD" и 
  нажать "Enter", установить пакеты "sounddevice" и "pyaudio" для питона:
     pip install sounddevice
     pip install pyaudio

4.Перейти в папку с cw_py, например: "cd c:\Users\Wasja\Desktop\cw_py_1.0.0",
  запустить для проверки командный файл "daily-cw.bat" - консоль очистится,
  будут выводиться на звуковую карту CW-слова 50 штук и печататься по 10 штук 
  в строке.

5.Можно редактировать "daily-cw.bat" на свой лад, или запускать программу без 
  командного файла прямо из консоли, например: 
      python cw_py.py --speed 120 --pause 3 --num 50 cw.txt
  
6.Для короткой помощи запустить: 
      python cw_py.py -h
